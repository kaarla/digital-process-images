La tarea1 se ejecuta desde la línea de comandos convert

$ python FiltrosKarla.py

al abrirse la interfaz gráfica, seleccionar del menú Imagen la opción Abrir
ya que esté cargada la imagen, selecionar el filtro que se desea aplicar del menú
filtros, la nueva imagen se guardará con un nombre aleatorio que hará referencia
al filtro que se aplicó

Para el filtro brillo no hay interfaz gráfica, para aplicarlo ejecutar:

$ python FiltroBrillo.py [imagen] [brillo] [nombre]

donde:
[imagen] corresponde al nombre de la imagen original que se desea modificar
[brillo] es el parámetro para aumentar o disminuir brillo, debe ser un entero positivo o negativo
[nombre] es el nombre con el que se desea guardar la imagen
