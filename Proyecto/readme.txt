Proceso Digital de Imágenes

Karla Socorro García Alcántara
Proyecto: Fotomosaico

Para ejecutar desde la línea de comandos de linux ejecute:

$ python Proyecto.py


Requerimientos: python, pip

Para instalar python en ubuntu:
$ sudo apt-get install python

Para instalar pip en ubuntu:
$ sudo apt-get install python-pip

Para generar su fotomosaico, de clic en el menú Foto Mosaico, seleccione la opción "Generar archivo de imágenes"
esta acción puede ser tardada, y al finalizar, de clic en la opción "Generar imagen", tambié tarda pero no tanto.

Se adjunta un ejemplo de la imagen conejo_mosaico.jpg generada a partir de la imagen conejo1.jpg; también la imagen comparacion.jpg que es una captura de la pantalla cuando se ejecutó el proyecto.
